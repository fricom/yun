<instructions>
a11y 에이전트를 사용하여 접근성 검사를 수행합니다.

## 사용법
```
/a11y [파일 경로 | Figma URL] [--level AA|AAA]
```

## 예시
```
/a11y src/pages/Login.tsx
/a11y https://figma.com/design/xxx?node-id=1:23 --level AAA
/a11y src/components/ --level AA
```

## 처리 흐름
1. 대상 파일/디자인 로드
2. WCAG 2.1 기준 레벨 확인 (기본값: AA)
3. 인지 가능성 → 운용 가능성 → 이해 가능성 → 견고성 순으로 검사
4. 위반 사항 심각도별 정렬 후 수정 방법 포함 보고서 반환
</instructions>
